Ashlar does not ask you to import a permanent runtime component package. The CLI
copies component source into your app, records what was installed, and leaves the
files available for review and customization.

## Integration rules

- Keep installed source in version control.
- Use `add --dry-run --diff` before large installs.
- Run `verify` after manual edits so the lockfile reflects intentional drift.
- Treat evidence status as the claim boundary. Visual polish is not proof.
