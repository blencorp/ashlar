The CLI is the main install surface. Run commands from your app root.

## Command map

```bash
pnpm dlx @blen/ashlar@latest --help
pnpm dlx @blen/ashlar@latest init
pnpm dlx @blen/ashlar@latest search button
pnpm dlx @blen/ashlar@latest suggest "Build a renewal case board"
pnpm dlx @blen/ashlar@latest add button --dry-run --diff
pnpm dlx @blen/ashlar@latest add button
pnpm dlx @blen/ashlar@latest verify
pnpm dlx @blen/ashlar@latest update button --dry-run
```

## Package manager forms

Use the one that matches your project:

```bash
pnpm dlx @blen/ashlar@latest add alert
npx @blen/ashlar@latest add alert
bunx @blen/ashlar@latest add alert
```

## Audit markup

```bash
pnpm dlx @blen/ashlar@latest audit --policy federal "src/**/*.{html,tsx,jsx,vue,svelte}"
```

Use `--policy components` for component-specific checks or `--policy all` when
you want both federal shell checks and installed-component checks.
