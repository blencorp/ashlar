Examples should prove integration, not decorate the repo. The current matrix is
a public site, docs app, Vite + Tailwind case board, vanilla TypeScript, React
SPA, Next.js, Svelte, and Vue.

## Start everything

```bash
pnpm testing:start
```

## One-shot verification

```bash
pnpm testing:start --check --visual
```

## Example quality bar

- Use real public-service workflows like case queues, renewal forms, document
  review, and approvals.
- Avoid one-off demo shells that do not exercise components and theme tokens.
- Do not use the old Vite experiment as the pattern for new examples.
- Every example should have install, audit, build, and visual smoke coverage.
